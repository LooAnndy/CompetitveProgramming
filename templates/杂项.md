#### 得到log2的上取整

```c++
i64 lg2(i64 x) {
    if (x == 0) return 0;
    return 64 - __builtin_clzll(x - 1LL);
}
```

#### sosdp
计算子集方案的和

```c++
for(int i=0;i<N;i++)
	for(int sta=(1<<N)-1;sta>=0;sta--)
	    if(sta&(1<<i))
			F[sta]+=F[sta^(1<<i)];
```

判断条件上加一个"!"即可计算超集

```c++
for(int i=0;i<N;i++)
	for(int sta=(1<<N)-1;sta>=0;sta--)
	    if(!(sta&(1<<i)))
			F[sta]+=F[sta^(1<<i)];
```
